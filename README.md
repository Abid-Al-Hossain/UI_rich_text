# UI Foundry Rich Text Editor Studio

Standalone, self-contained Rich Text Editor component generator built for marketplace sale and future UI Foundry SaaS merge-readiness.

## Features

- Workflow model: groups, actions, values, saved state, dirty/reset previews
- Field systems: labels, validation, shortcut hints, item metadata, keyboard navigation
- Composition: sections, panels, command lists, filters, editor blocks, draggable items
- Accessibility: roles, announcements, focus order, reduced motion and escape behavior
- React-only export: preview, visible code, copy, and download are sourced from one payload.
- Local shared controls: no runtime imports from other component projects.

## Commands

```bash
npm install
npm run typecheck
npm run lint -- --quiet
npm run build
npm run dev
```
